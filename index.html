FAQ

Common questions.

Does it work on every website?
PagePilot Capture works on the vast majority of websites. Chrome system pages (like chrome://settings) cannot be captured due to browser security restrictions — the extension will show a friendly error if you try. Pages behind login walls are captured exactly as you see them when logged in.
Will it capture lazy-loaded images?
Yes. The extension adds a settle delay after each scroll step so lazy-loaded content has time to render before the screenshot fires. Very slow CDNs or custom loaders may occasionally not fully resolve — in those cases a second capture pass usually helps.
How does the PDF export work?
The full-page canvas is sliced into A4-sized rows and each row is embedded as a JPEG image in a standard PDF. No external library is required. The resulting PDF downloads directly to your machine — nothing is sent to a server.
Is my data stored or sent anywhere?
No. All processing happens locally inside your browser using the Canvas API. Screenshots are stored temporarily in Chrome's session storage until the preview tab loads, then immediately deleted. No analytics, no telemetry, no servers. See our Privacy Policy for full details.
Can I change the keyboard shortcut?
Yes. Go to chrome://extensions/shortcuts in Chrome, find PagePilot Capture, and assign any combination you prefer. The default is Alt+Shift+P.
Is it really free?
Yes, completely free. No freemium tier, no capture limits, no watermarks. Install it and use it without restriction.
